# Blog mobile application

Blog application on mobile devices created to cooperate with 
Blog DRF API.

### You can check this application here
[Application](https://expo.io/@michal-mietus/Blog)

### Features
- login
- register
- adding posts
- viewing posts

### Technologies
- React Native
- tcomb-form-native
- react-navigation
- react-native-loading-spinner-overlay
- Expo (for developing)
