# Blog mobile application


Application created for mobile devices to cooperate with Blog 
DRF REST Api.

### You can check this application here
[Application](https://expo.io/@michal-mietus/Blog)

### Features
- login
- registration
- adding articles
- viewing articles

### Technologies
- React Native
- tcomb-form-native
- react-navigation
- react-native-loading-spinner-overlay
- Expo (for developing)
